CSS Effects: Outline, Shadow, Gradient & Units

This box has an outline effect with a shadow around it. The outline is a distinct border surrounding the actual border of the box. You can use `outline` in CSS to create emphasis or focus on elements.

Notice the use of different units here, such as rem for relative font sizing, px for fixed dimensions, and % for responsive layouts. The box above uses a gradient background, and the button uses both gradient and shadow effects.
